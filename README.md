## Setup & Run
```Bash
# setup
>> npm install

# compile (Precautions for compile)
>> truffle compile

# migrate
>> truffle migrate

# run
>> npm run dev
```


## ❗️Precautions for compile

Each contracts needs to be compiled to suit the solidity version (refer to `truffle-config.js`).
